# Example 2 — Line chart with color-pivot: Monthly Sales by Segment

## Tableau

```xml
<worksheet name="Monthly Sales by Segment">
  <table>
    <view><datasources><datasource name="Orders"/></datasources></view>
    <rows>[federated.x].[sum:Sales:qk]</rows>
    <cols>[federated.x].[mdy:Order Date:ok]</cols>
    <encodings>
      <color column="[federated.x].[Segment]"/>
    </encodings>
    <mark class="Line"/>
    <filter column="[Order Date]" class="relative-date" period-type-v2="last-n-months" first-period="-12" last-period="0"/>
  </table>
</worksheet>
```

## dbt model addition

In `models/orders.yml` ensure `order_date` is a timestamp dimension with intervals:

```yaml
- name: order_date
  meta:
    dimension:
      type: timestamp
      label: Order Date
      time_intervals:
        - DAY
        - WEEK
        - MONTH
        - QUARTER
        - YEAR
- name: segment
  meta:
    dimension:
      label: Segment
      type: string
```

## Lightdash chart

```yaml
chartConfig:
  config:
    eChartsConfig:
      legend:
        show: true
      series:
        - encode:
            xRef:
              field: orders_order_date_month
            yRef:
              field: orders_total_sales
          showSymbol: true
          smooth: false
          type: line
    layout:
      xField: orders_order_date_month
      yField:
        - orders_total_sales
  type: cartesian
contentType: chart
metricQuery:
  dimensions:
    - orders_order_date_month
    - orders_segment
  exploreName: orders
  filters:
    dimensions:
      and:
        - id: f-relative-12mo
          operator: inThePast
          settings:
            completed: false
            unitOfTime: months
          target:
            fieldId: orders_order_date
            tableName: orders
          values:
            - 12
  limit: 500
  metrics:
    - orders_total_sales
  sorts:
    - descending: false
      fieldId: orders_order_date_month
name: Monthly Sales by Segment
pivotConfig:
  columns:
    - orders_segment
slug: monthly-sales-by-segment
spaceSlug: sales
tableName: orders
version: 1
```

Notes:
- Tableau month-truncation `mdy:Order Date` → Lightdash `orders_order_date_month` (auto-generated by `time_intervals: [..., MONTH]`).
- Color-by-Segment → `pivotConfig.columns: [orders_segment]`.
- Relative-date filter → `inThePast` with `unitOfTime: months`.
